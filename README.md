# DeepRacer DeepStats

Welcome! The code here consists of:

1. A CloudFormation `.yaml` file to set up a SageMaker Notebook environment
2. Some `.ipynb` files (Jupyter Notebooks) which can help you analyze DeepRacer training and evaluation data

The notebook files borrow heavily from content that is already available. In particular [this notebook from AWS](https://github.com/aws-deepracer/aws-deepracer-workshops/tree/master/log-analysis). 

I will list sources here in the README, and possibly within the notebook files themselves, where it aids clarity. 

Happy Racing!
